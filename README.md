# MinWin32Exec2024

2024-11-24現在、Win32 x64構成で最小の実行可能バイナリを作成する実験。

## 方針

- Cランタイムライブラリの排除
- セキュリティチェックオプションの無効化

## 道具立て

- Visual Studio 2022 (v143)
- Windows SDK 10.0.26100.0

## プロジェクト設定

### C/C++

| option | value |
| - | - |
| C++の例外を有効にする(/EH) | いいえ |
| SDLチェック(/SDL) | いいえ(/sdl-) |
| セキュリティチェック(/GS) | 無効(/GS-) |
| 最適化(/O) | 最大最適化(サイズを優先)(/O1) |

### リンカー

| option | value |
| - | - |
| マニフェストの生成(/MANIFEST) | いいえ(/MANIFEST:NO) |

## 結果

|バイナリ|サイズ (bytes)|
|-|-|
|MinWin32Con.exe|3072|
|MinWin32Gui.exe|2560|
